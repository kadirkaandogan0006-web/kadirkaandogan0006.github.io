# 📂 Resmi Proje Depolama ve Web Portfolyosu

Geliştirilen uygulamaların kaynak kodlarının, sıkıştırılmış dosyalarının barındırıldığı ve canlı web tabanlı simülasyon araçlarının yer aldığı resmi GitHub deposudur.

---

## 🛠️ İçerik ve Özellikler

### 📦 Proje Dosyaları (İndirme Alanı)
Web sitesi üzerinden erişilebilen projeler kısmından, bu depoda saklanan sıkıştırılmış proje dosyaları doğrudan indirilebilir durumdadır. Dosyalar her güncelleme ile birlikte otomatik olarak yenilenir.

### 🧪 Canlı Laboratuvar Paneli
Web arayüzünde yer alan tarayıcı tabanlı mini hücre popülasyonu motorudur:
* **Hücre Sayısı Kontrolü:** Sürgülü panel yardımıyla ekrandaki hücre popülasyonu anlık değiştirilebilir.
* **Mitoz Tetikleyici:** Tek bir işlemle ekrandaki hücrelerin tamamı gerçek zamanlı bölünme döngüsüne sokulabilir.
* **Sekme Algoritması:** Hücrelerin ekran dışına çıkmasını engelleyen sınır kontrol algoritmaları mevcuttur.

---

## 💻 Teknolojik Altyapı

* **Arayüz:** HTML5 semantik etiket yapısı ve CSS3 Grid mimarisi (Karanlık tema odaklı modern tasarım).
* **Simülasyon Motoru:** JavaScript (ES6+) Nesne Yönelimli Programlama (OOP Class) mimarisi ile kurulmuş bağımsız hücre simülasyon döngüsü.
